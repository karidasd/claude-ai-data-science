Data Science MCP Server Configuration

Give Claude Desktop or Claude Code the power to execute python code and analyze local datasets.

claude_desktop_config.json 
{
  "mcpServers": {
    "data-science-mcp": {
      "command": "python",
      "args": [
        "C:/Users/karid/.gemini/antigravity/scratch/claude-ai-data-science/mcp/ds_mcp_server.py"
      ],
      "env": {
        "PYTHONUNBUFFERED": "1"
      }
    }
  }
}

How to set this up:

  1. Press Win + R, type %APPDATA%\Claude and hit Enter.
  2. Open or create claude_desktop_config.json in a text editor.
  3. Copy the JSON above and paste it inside the file, then save.
  4. Restart your Claude Desktop application.

Trigger Criteria

Automatically triggered when working on python data science scripts, CSV files, pandas, numpy, scikit-learn, and matplotlib.

Key Guidance Rules

  • Distribution Checking: Always verify statistics with describe() and null counts.
  • Data Leakage Prevention: Perform train-test splits before running standard scaler transformations.
  • Target Imbalance Management: Check if targets are skewed; suggest stratified splits or class weights.
  • High Quality Plotting: Labels, titles, gridlines, and high-DPI saving are mandatory.
